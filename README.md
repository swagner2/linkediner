## üìã The Four Core Principles

Every LinkedIn post MUST be:

1. **Honest** - Sounds like a real person's real experience
2. **Clear** - ONE obvious main point
3. **Useful** - Worth the reader's time
4. **Simple** - 5th grade reading level

If all four aren't met, the post needs revision.

## üí° Pro Tips

### For Hooks:
- Lead with your most surprising result
- Use exact numbers, not ranges
- Make counterintuitive claims

### For Content:
- Show, don't tell
- Include small failures
- Name specific tools/resources
- Add timestamps

### For Format:
- Line break every 1-2 sentences
- Use arrows (‚Üí) for steps
- No paragraph over 3 lines
- Create white space
